    • O que é HTML ?

    HTML é a sigla de HyperText Markup Language, expressão inglesa que significa "Linguagem de Marcação de Hipertexto".

    Veja como é uma estrutura básica de HTML:

    O HTML serve para desenvolver a estrutura de uma página ou conteúdo digital. Na imagem acima, você pode ver um aglomerado de instruções que foram digitadas em um programa (respeitando a sintaxe do HTML) para que os componentes da página apareçam de uma forma específica nela.
    Todas as palavras em rosa são chamadas de TAG. No entanto, as palavras em verde são os atributos (ou valores) dessa TAG.
    Perceba que a TAG é formada pelos sinais “<” e “>”
    Exemplo de ABERTURA < p >

    e FECHAMENTO < /p >


    Veja que a ABERTURA é simples, contém apenas a estrutura básica com o nome da TAG P, que significa Parágrafo, já no FECHAMENTO a tag é replicada, porém com o sinal de "/"

     antes do nome. Outro fechamento que é menos comum, é o “< >” (sem a barra).

    Por exemplo:


    A Tag meta tem o atributo name, que por sua vez permite que você escolha um nome para que ele possa ser usado para se comunicar com outros navegadores ou outros tipos de instruções. O atributo classe por exemplo, pode ter o mesmo nome como o valor de várias TAGS diferentes, permitindo que você faça uma edição única para todos os itens que estiverem com essa classe definida. Seria muito difícil trabalhar por exemplo editando TAG por TAG sendo que com a classe, você pode englobar todas em um só nome e editá-las todas de uma só vez com apenas um comando.

    META

    significa metadados, que são informações que vão descrever os dados que contém em seu arquivo. Já o "UTF-8" (UCS Transformation Format 8) é um tipo de codificação utilizada para comunicar aos navegadores qual será o formato da codificação dos caracteres utilizados naquele arquivo.
    Todas as informações mostradas acima, compõem a parte oculta da página, ela serve apenas para configurações internas.

    TAG Body, para que serve ?

    A palavra Body em inglês significa "corpo". Essa TAG no HTML é a base da construção do seu site. É dentro dela que você deve incluir todo o conteúdo da sua página.
